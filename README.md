# Presentacion Gobierno Tlaxcala

Repositorio de materiales para una demostracion de analisis documental aplicado a procesos administrativos de gobierno.

## Contenido

- `presentacionGobTlaX_v2.pptx`: presentacion principal.
- `Taller_Proyectos_Sinteticos/`: expedientes sinteticos para ejercicios de seguimiento institucional.
- `Images/`: recursos visuales utilizados en la presentacion.
- `PROMPTS_DEMO.md`: conjunto de prompts para evaluar reconstruccion de procesos, trazabilidad, responsables, decisiones y generacion de datasets.

## Uso sugerido

Usar los expedientes sinteticos como carpeta documental de entrada y ejecutar los prompts incluidos en `PROMPTS_DEMO.md` para demostrar capacidades de analisis institucional, trazabilidad administrativa y estructuracion de informacion.
